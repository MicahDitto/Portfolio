Development

In high school, I took an AP Computer Science course that sparked my interest in Java and other coding languages. Since then I have made giant strides in my skills as a developer. My early experience at Penn State buffeted my defensive programming skills by teaching me how to better test and debug software applications. I quickly learned how to use many other techniques and structures such as class hierarchy, Java interfaces, GUIs, collections, to take my Java to the next level. However, knowing that Java programming is brittle and has its weaknesses, I expanded my knowledge of other programming languages. I have really dug into JavaScript and HTML since then, but I also practice working with Python, PHP, and CSS. I have also trained my skills using SQL to manage databases. Working effectively both on my own and in groups, I am now capable of using Representational State Transfer or RESTful design pattern, Model-View-Controller (MVC) architectural for Java and web frameworks, databases, APIs, and client/server computing to create complex programs, applications, and websites. Some of the applications that display my ability in development are my Student Grade Predictor in Java, Flix2You database redesign using SQL and several other websites that can be viewed under the Development Examples page.

Family

I am always excited to talk about my family because it usually prompts some great reactions. I have seven siblings, two brothers, and five sisters, so my family is best classified as a “Big” one. With ten people in the house, it’s always a party. I really enjoy being in a big family. It has taught me many wonderful principles that I feel other families might miss. I have learned to be very flexible and adaptive to situations as I often needed to prioritize family needs over my own wants. And as the second oldest responsibility and care for others is something I value. My dad has exceptionally modeled hard work, as it is no easy task having so many dependants. My father once told me “You are always leading people, but what are you leading them towards?” This I have resonated with for years as it taught me to recognize when to be a strong leader, and when to be a good follower. Overall, they are best described as a crazy blessing.
